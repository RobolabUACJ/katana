**********************************************************************************
Instalation of the player plug-in
**********************************************************************************
-Copy the folder ssc32 to your home path

	1.$ cp -r ssc32 ~/

-Follow the next steps

	2.$ cd ~/ssc32

	3.$ mkdir build

	4.$ cd build/

	5.$ cmake ../

	6.$ make

-Run the server program to make sure you installed the plug-in correctly

	7.$ cd ../
	8.$ player samurai_ssc32.cfg
